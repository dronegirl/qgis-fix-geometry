# qgis-fix-geometry
This repository contains a Python script designed for use in QGIS that automatically fixes common geometry errors in vector layers. It is particularly useful for cleaning up and preparing geospatial data for analysis and visualization.
Features
Geometry Validation: Identifies and corrects issues such as invalid geometries and multipart geometries.
User-Friendly: Simple to run directly from the QGIS Python console.
Customizable: Easy to modify the script to suit specific geometry fixing needs.
Usage
Load your vector layer in QGIS.
Run the provided script in the QGIS Python console to fix geometry issues within the layer.
Requirements
QGIS
